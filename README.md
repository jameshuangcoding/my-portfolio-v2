# huangjames.dev

Personal portfolio website built with Next.js, featuring a minimalist design with dark mode support and a custom neon orange accent palette.

## Tech stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Icons:** React Icons
- **Deployment:** Vercel

## Getting started

Node.js 20+ required.

```bash
git clone https://github.com/jameshuangcoding/my-portfolio-v2.git
cd my-portfolio-v2
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Project structure

```
my-portfolio-v2/
├── components/       # React components (About, Experience, Projects, NavBar, Footer)
├── data/             # Content files (projects, experiences, links)
├── public/           # Static assets
└── src/app/          # Next.js app directory (layout, page, globals.css)
```
